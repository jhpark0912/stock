## 🛠️ 실행 스크립트

프로젝트 루트에 다음 실행 스크립트가 있습니다:

| 파일 | 설명 | 사용 방법 |
|-----|------|---------|
| `start_all.bat` | 전체 실행 (백엔드 + 프론트엔드) | 더블클릭 |
| `start_backend.bat` | 백엔드만 실행 | 더블클릭 |
| `start_frontend.bat` | 프론트엔드만 실행 | 더블클릭 |
| `create_deployment_package.py` | 배포 패키지 생성 | `python create_deployment_package.py` |
